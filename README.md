# IOTO Update

This plugin is used to update the IOTO Framework, deploy AI Agent skills, install companion plugins, and sync your personal templates — all from within Obsidian.

## Features

### IOTO Framework Updates

Sync content from Airtable/NocoDB into your vault:

- **Update Core Files** — Update the IOTO Framework core files
- **Update Help Docs** — Update help documentation (with date filtering)
- **Update MYIOTO Templates** — Update MYIOTO templates
- **Update CSS Snippets** — Update CSS snippets to `.obsidian/snippets`
- **Update IOTO Framework Setting Plugin** — Update the IOTO Settings plugin

### One-Click Deploy

- **Deploy IOTO With One Click** — Run all framework updates in a single command

### AI Agent Skills

- **Update Skills** — Deploy AI Agent skills to your preferred platform. You will be prompted to select a target platform:

    | Platform     | Folder      |
    | ------------ | ----------- |
    | TRAE         | `.trae`     |
    | Claude Code  | `.claude`   |
    | OpenCode     | `.opencode` |
    | OpenAI Codex | `.agents`   |
    | Mimo Code    | `.mimocode` |
    | Hermes       | `.hermes`   |

### Personal Sync Templates

- **Get Your Personal Sync Templates** — Fetch your personal sync templates from your own Airtable base
- **Update User Permissions** — Refresh your user permissions and update IDs

### Plugin Installation

Install companion plugins directly from GitHub or Gitee:

- **Install IOTO Template Generator**
- **Install IOTO Dashboard**
- **Install IOTO Tasks Center**
- **Install Sync Scripts Generator** (available for specific users)

### Self-Update

- Check for plugin updates and install the latest version from the settings panel

## Settings

### Basic Settings

- **Update API Key** — Your license API key (validated)
- **Email Address** — The email used at purchase (validated)
- **Plugin Download Source** — Choose between GitHub and Gitee
- **IOTO Running Language** — Auto (follow Obsidian language), Simplified Chinese, Traditional Chinese, or English
- **IOTO Framework Path** — Path to the IOTO Framework folder in your vault

### User Sync Settings

- **Airtable Personal Token** — For accessing your personal sync base
- **Sync Setting URL** — Airtable URL of your sync configuration table
- **Sync Templates Folder** — Destination folder for personal sync templates

## Plugin Download Source

All plugin downloads and updates can be configured to use either **GitHub** or **Gitee**, making it accessible for users in different regions.

## Languages

- English
- Simplified Chinese (简体中文)
